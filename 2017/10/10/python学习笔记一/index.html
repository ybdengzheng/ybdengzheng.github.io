python学习笔记一

发表于 2017-10-10 |

历史

python是一个用C开发的语言，设计哲学“优雅”、“明确”、“简单”。

版本

Python 2.x官方支持到2020年，意味着新的项目将更多采用Python3.x来进行开发，以便于日后的维护和支持。但是由于3.x并不支持2.x，现有2.x的项目仍然会采用2.x来进行维护和更新。并且2.x在现阶段拥有更广泛的插件支持。所以作为学习，我会主要以Python3.x作为基础来学习，但是也会了解一些2.x的内容来解决可能存在的某些兼容性问题。

安装python

准备工作

Python for Mac OS的下载地址，下载适合你的Python版本。Homebrew是一个很方便的快速安装一些常用的工具。安装gcc编译器，可以使用xcode-select --install来进行安装，当然前提是你已经装好Xcode 或者xcode command line tool 等基础工具

安装Ptyhon3

[localhost:localhost] brew install python3

几分钟以后homebrew将会自动安装好pip3.

运行

localhost@localhost /u/l/H/bin> python --version
Python 2.7.13
 localhost@localhost /u/l/H/bin> pip --version
 pip 9.0.1 from /usr/local/lib/python2.7/site-packages (python 2.7)
localhost@localhost /u/l/H/bin> python3 --version
Python 3.6.3
localhost@localhost /u/l/H/bin> pip3 --version
pip 9.0.1 from /usr/local/lib/python3.6/site-packages (python 3.6)
